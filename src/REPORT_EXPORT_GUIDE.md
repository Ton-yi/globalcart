# 报表导出功能说明

## 功能概述

数据报表系统现已支持将统计数据导出为 **Excel (.xlsx)** 或 **CSV** 格式文件，方便财务核算和数据存档。

## 使用方式

### 导出步骤

1. 进入 **数据报表** 页面（`/AdminReports`）
2. 选择需要导出的时间范围（开始日期 - 结束日期）
3. 可选：设置维度、粒度、对比等筛选条件
4. 点击右上角的 **导出 Excel** 或 **导出 CSV** 按钮
5. 文件将自动下载到本地

### 导出格式说明

#### Excel (.xlsx) - 推荐
- **多工作表结构**，包含：
  - **汇总数据**：核心财务和订单指标
  - **时间序列**：按日/周/月的趋势数据
  - **维度明细**：按选定维度（如订单状态、目的地国家等）的分组数据
  - **对比分析**：如果启用了对比功能，包含同比/环比数据
- **适用场景**：财务核算、深度数据分析、正式报告

#### CSV (.csv)
- **单文件纯文本格式**
- **兼容性强**，可在 Excel、Google Sheets、Numbers 等工具中打开
- **适用场景**：数据导入其他系统、快速查看、程序化处理

## 导出内容

### 1. 汇总数据

| 指标 | 说明 |
|------|------|
| 下单收入 (JPY) | 用户下单阶段实际支付金额（含商品货款 + 服务费） |
| 发货收入 (JPY) | 发货阶段收取的运费收入 |
| 总利润 (JPY) | 下单利润 + 发货利润 |
| 下单利润 (JPY) | 下单收入 - 退款 - 商品成本 |
| 发货利润 (JPY) | 运费收入 - 国际运费 - 外箱成本 |
| 服务费收入 (JPY) | 代购服务费收入 |
| 增值服务收入 (JPY) | 打包/验货/拍照等收入 |
| 退款合计 (JPY) | 期间退款总金额 |
| 商品成本 (JPY) | 商品采购成本 |
| 订单总数 | 期间创建的订单总数 |
| 活跃客户数 | 期间有下单的客户数（去重） |
| 新客户数 | 期间首次下单的客户数 |
| 平均订单金额 (JPY) | 客单价 |
| 待付款订单数 | 待确认/待付款订单 |
| 待采购订单数 | 订单状态为待采购 |
| 待发货订单数 | 已入库/仓储中订单 |

### 2. 时间序列数据

按选择的粒度（日/周/月）展示趋势数据：
- 收入 (JPY)
- 利润 (JPY)
- 服务费 (JPY)
- 运费收入 (JPY)
- 增值服务 (JPY)
- 订单数
- 退款 (JPY)

### 3. 维度明细数据

按当前选择的维度（如订单状态、目的地国家、运输方式等）分组展示：
- 维度值
- 订单数
- 收入 (JPY)
- 利润 (JPY)
- 服务费 (JPY)
- 退款 (JPY)

### 4. 对比分析（如启用对比功能）

- 当前期间数值
- 对比期间数值
- 变化率（百分比）

## 权限要求

- **管理员**（admin、tenant_admin）
- **工作人员**（staff）
- **平台管理员**（platform_admin）

普通用户（user）无法访问报表页面，因此也无法导出数据。

## 文件命名规则

```
报表导出_{开始日期}_至_{结束日期}.{格式}
```

示例：
- `报表导出_2026-05-01_至_2026-05-31.xlsx`
- `报表导出_2026-06-01_至_2026-06-11.csv`

## 技术实现

### 前端
- 页面：`pages/AdminReports.jsx`
- 导出按钮调用后端函数 `exportReportData`
- 自动下载生成的文件

### 后端
- 函数：`functions/exportReportData.js`
- 使用 `xlsx` 库生成 Excel 文件
- CSV 使用原生 JavaScript 生成
- 数据来源于 `getReportData` 函数

### 依赖
- `xlsx@0.18.5` - Excel 文件生成

## 注意事项

1. **数据时效性**：导出的是当前筛选条件下的实时数据
2. **数据范围**：仅导出当前租户的数据（租户隔离）
3. **文件大小**：大量数据（如超过 10,000 行）可能导致导出时间较长
4. **浏览器兼容性**：需要现代浏览器支持 Blob 和 download 属性
5. **中文编码**：CSV 文件使用 UTF-8 编码，Excel 可直接识别

## 常见问题

### Q: 导出的 Excel 文件打开是乱码？
A: Excel 文件是标准格式，不会出现乱码。如果是 CSV 文件，请在 Excel 中使用"数据导入"功能，选择 UTF-8 编码。

### Q: 导出失败怎么办？
A: 检查以下几点：
- 是否有管理员权限
- 选择的时间范围内是否有数据
- 网络连接是否正常

### Q: 可以自定义导出字段吗？
A: 当前导出所有标准字段。如需自定义，请联系管理员修改后端函数。

### Q: 导出的数据为什么和页面显示不一致？
A: 确保导出时页面筛选条件和导出参数一致。导出功能使用的是相同的后端数据源。

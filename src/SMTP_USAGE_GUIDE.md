# SMTP 功能使用指南

## 🚀 快速开始

### 1. 配置 SMTP 发信

#### 步骤
1. 管理员登录 → AdminSettings → 通知设置
2. 启用"启用 SMTP 发信"
3. 填写 SMTP 配置：
   - **SMTP 服务器**: 如 `smtp.gmail.com`
   - **端口**: 推荐 `587` 或 `465`
   - **用户名**: 邮箱地址
   - **密码**: 邮箱服务商提供的授权码
   - **发件人名称**: 如"同一物流通知中心"
   - **发件人邮箱**: 通知邮箱地址
4. 点击"发送测试邮件"验证
5. 保存设置

### 2. 常见邮箱配置

#### Gmail
```
SMTP 服务器：smtp.gmail.com
端口：587
SSL/TLS: 开启
用户名：your@gmail.com
密码：[应用专用密码]
```
**获取授权码**: 
1. 开启两步验证
2. 访问 https://myaccount.google.com/apppasswords
3. 创建应用专用密码

#### QQ 邮箱
```
SMTP 服务器：smtp.qq.com
端口：587
SSL/TLS: 开启
用户名：your@qq.com
密码：[SMTP 授权码]
```
**获取授权码**:
1. 设置 → 账户
2. 开启 POP3/SMTP 服务
3. 获取授权码

#### 163 邮箱
```
SMTP 服务器：smtp.163.com
端口：587
SSL/TLS: 开启
用户名：your@163.com
密码：[SMTP 授权码]
```

#### Outlook
```
SMTP 服务器：smtp-mail.outlook.com
端口：587
SSL/TLS: 开启
用户名：your@outlook.com
密码：[应用密码]
```

## 📋 功能说明

### 支持的发送方式

#### 1. SMTP 发信（推荐）
- 优先级最高
- 支持自定义发件人
- 支持所有邮箱服务商

#### 2. Gmail OAuth
- 需要授权 Gmail 连接器
- 自动续期
- 安全性高

#### 3. 平台默认
- 使用 Base44 平台邮件服务
- 无需配置
- 适合初期使用

### 自动触发场景

以下场景会自动发送邮件：
- 订单状态变更
- 付款通知
- 发货通知
- 拼邮通知
- 消息通知

### 用户偏好设置

用户可在"个人设置 → 通知设置"中：
- 开启/关闭邮件通知
- 按通知类型设置
- 仅接收重要通知

## 🛠️ 故障排查

### 测试邮件发送失败

#### 错误："SMTP 配置不完整"
**原因**: 缺少必填字段  
**解决**: 检查 SMTP 服务器、用户名、密码是否填写

#### 错误："不支持的 SMTP 端口"
**原因**: 端口不在白名单内  
**解决**: 使用支持的端口：25, 465, 587, 2525

#### 错误："认证失败"
**原因**: 密码错误或邮箱服务商阻止  
**解决**:
1. 确认使用授权码而非登录密码
2. 检查邮箱服务商的安全设置
3. 允许"不够安全的应用"（如 Gmail）

#### 错误："连接超时"
**原因**: 网络问题或服务器不可达  
**解决**:
1. 检查 SMTP 服务器地址
2. 确认端口正确
3. 检查防火墙设置

### 通知未发送

#### 检查清单
1. 用户是否开启了邮件通知
2. SMTP 配置是否正确
3. 测试邮件是否能收到
4. 检查通知偏好设置

## 🔐 安全提示

### 重要提醒
1. **不要共享密码** - 仅管理员知晓
2. **使用授权码** - 避免使用登录密码
3. **定期更换** - 建议每 90 天
4. **监控异常** - 发现异常立即处理
5. **启用加密** - 始终使用 SSL/TLS

### 泄露应对
如果发现密码泄露：
1. 立即更换邮箱密码
2. 更新 SMTP 配置
3. 检查发送记录
4. 通知受影响用户
